# 🛍️ Zudio Sales Analysis (2025)

## 📊 Project Overview

This project is an end-to-end data analytics workflow built using retail sales data inspired by Zudio. It focuses on analyzing business performance, identifying trends, and generating actionable insights using SQL, Excel, and Power BI.

---

## 🎯 Objectives

* Analyze sales performance across different categories and cities
* Identify top-performing products and revenue drivers
* Track monthly sales trends
* Build interactive dashboards for business decision-making

---

## 🛠️ Tools & Technologies

* **MySQL** – Database design, data storage, and querying
* **Microsoft Excel** – Data cleaning, analysis, and dashboard creation
* **Power BI** – Interactive data visualization and reporting

---

## 🔄 Project Workflow

1. Designed a relational database schema using MySQL
2. Inserted and managed 100+ realistic sales records
3. Performed SQL analysis using joins, aggregations, and filters
4. Exported cleaned dataset to Excel for further analysis
5. Built an Excel dashboard using Pivot Tables, charts, and KPIs
6. Developed an interactive Power BI dashboard for advanced visualization

---

## 📈 Key Insights

* Identified top-performing product categories contributing to revenue
* Analyzed city-wise sales distribution and performance trends
* Discovered monthly revenue patterns and growth trends
* Evaluated customer purchasing behavior through order analysis

---

## 📊 Dashboard Features

* KPI Cards (Total Revenue, Total Orders, Average Order Value)
* Revenue breakdown by category, city, and product
* Monthly sales trend analysis
* Interactive filters (slicers) for dynamic insights
